# OpenFood

A delivery system API like iFood written in Java.

## Developed with

- Java 17
- Domain Driven Design
- Domain Events
- Spring Boot 3.0.0
- Lombok
- MySQL
- Hibernate
- Flyway
- REST Assured
- ModelMapper
- JasperReports
- AWS SDK with Amazon S3
- JavaMail with Amazon SES
- Apache FreeMarker
- Shallow ETags
- REDIS
