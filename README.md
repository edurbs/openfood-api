# OpenFood

A delivery system API like iFood written in Java.

## Developed with

- Java 17
- Domain Driven Design
- Domain Events
- Spring Boot 2.7.6
- Lombok
- MySQL
- Hibernate
- Flyway
- REST Assured
- ModelMapper
- JasperReports
- AWS SDK with Amazon S3
- JavaMail with Amazon SES
- Apache FreeMarker
- Shallow ETags
- REDIS
- OpenAPI 3 with Springdoc (swagger)
